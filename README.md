# Fintech Ops Platform

A complete Financial Operations, Reconciliation, Risk Analysis and Business Analytics platform simulating the internal systems of a global fintech.

This repository was designed to demonstrate hands-on experience aligned with Business Analyst (Finance), Fintech Operations and Product Analyst roles.

## Elevator Pitch (60s)
I built a fintech operations platform that simulates transaction processing, ledger accounting, external settlements, reconciliation workflows, and risk analysis. The project reflects how modern digital banks operate internally and how analysts monitor financial integrity, performance KPIs, and regulatory risks using data.

## Architecture
User Payments → Transactions → Ledger → External Settlement → Reconciliation → Risk Analysis → KPIs

## Skills Demonstrated
- Financial operations & reconciliation
- SQL analytics and data modeling
- Risk & anomaly detection
- Business documentation
- Fintech domain knowledge
